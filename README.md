> **Revolutionizing student living through data-driven compatibility and modern automation.**

**Hostel lite** is a streamlined, full-stack ecosystem designed to replace bulky legacy systems with a high-performance interface. While traditional management tools focus purely on administrative tasks, **Hostel lite** introduces an intelligent **Smart Roommate Matchmaking Engine**. This system leverages lifestyle data to ensure students are paired with compatible roommates, fostering a more harmonious and productive academic environment.

---

## 🚀 The Standout Feature: Smart Matchmaking Engine

This project moves beyond random room allocation by implementing a custom logic that pairs students based on critical compatibility vectors:

* **⏰ Chronotype Alignment:** Syncs sleep schedules to prevent disruptions between "Night Owls" and "Early Birds."
* **📚 Academic Sync:** Groups students with similar study habits and quiet-hour requirements.
* **🧹 Lifestyle Standards:** Filters based on cleanliness preferences and social habits to minimize domestic friction.

---

## 🛠️ Technical Stack & Architecture

Designed for scalability and performance, **Hostel lite** utilizes a modern, multi-tier architecture:

* **Frontend:** Built with **React** and **Tailwind CSS**, featuring a minimalist **glassmorphism** aesthetic and fluid **Framer Motion** animations.
* **Backend:** A robust hybrid of **Node.js** and **PHP** to handle complex API logic and real-time requests.
* **Database:** A normalized **MySQL** schema optimized for relational queries between students, rooms, and preference data.
* **Security:** Implementation of advanced **cryptographic modes** to ensure the privacy and integrity of student records.

---

## 💎 Core Modules

### **🛡️ Administrator Dashboard**
* **Dynamic Inventory:** Real-time tracking of room availability, floor layouts, and occupancy status.
* **Automated Allocation:** Streamlined student check-in/check-out with digital record-keeping.
* **Financial Tracking:** Integrated dashboard for monitoring fee payments and generating digital receipts.

### **🎓 User Portal**
* **Digital Onboarding:** Simplified registration and automated room applications.
* **Preference Profiling:** A behavioral data-entry point that powers the Matchmaking Engine.
* **Community Hub:** Access to digital notice boards and a streamlined grievance redressal system.

---

## 📂 Project Structure

```bash
Hostel-lite/
│
├── backend/            # Node.js and PHP APIs/Logic
├── frontend/           # React components and Tailwind styles
├── database/           # MySQL schema and data migrations
└── Screenshots/        # Visual documentation of the glassmorphism UI
```

## **Server Configuration:**
    *   Place the project folder in your `C:/xampp/htdocs/` directory.
    *   Start **Apache** and **MySQL** via the XAMPP Control Panel.
3.  **Database Integration:**
    *   Import the provided SQL file in the `/database/` folder into your local `phpMyAdmin` environment.
4.  **Run the Application:**
    *   Navigate to `http://localhost/HMS` in your browser.
