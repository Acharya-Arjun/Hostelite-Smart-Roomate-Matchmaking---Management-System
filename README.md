# Hostel Management System with Smart Roommate Matching System

## Overview

The Hostel Management System with Smart Roommate Matching System is a full-stack web application designed to simplify hostel administration and improve the student accommodation experience.

The system helps administrators manage hostel operations efficiently while also providing an intelligent roommate matching feature based on student preferences and compatibility.

---

# Features

## Admin Module

* Manage hostel rooms and availability
* Add, update, and delete student records
* Manage room allocations
* Track hostel occupancy
* Monitor hostel data through dashboard analytics

## Student/User Module

* Student registration and login
* View hostel room details
* Apply for hostel accommodation
* View allocated room details
* Update profile information

## Smart Roommate Matching System

* Preference-based roommate allocation
* Matching based on:

  * Study habits
  * Sleeping schedule
  * Cleanliness preferences
  * Lifestyle compatibility
* Improves student comfort and hostel experience

# Installation

## Clone Repository

```bash
git clone https://github.com/your-username/hostel-management-smart-roommate-system.git
```

## Move Project to XAMPP htdocs

```bash
C:/xampp/htdocs/
```

## Start Services

* Start Apache
* Start MySQL

using XAMPP Control Panel.

## Import Database

1. Open phpMyAdmin
2. Create a database
3. Import the provided SQL file

## Run Project

Open in browser:

```bash
http://localhost/HMS
```
